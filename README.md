# Heart Attack Predictor 🩺

An AI-powered web application that uses Deep Learning to predict the likelihood of a heart attack based on medical parameters.

## 🚀 Features
* **Deep Learning Model:** Built using TensorFlow/Keras (`model.h5`).
* **Web Interface:** Interactive UI built with Flask.
* **Real-time Prediction:** Users can input data and get instant results.

## 📁 Project Structure
* `app.py`: The main Flask application.
* `model_training.py`: Script used to train the neural network.
* `templates/`: HTML files for the web interface.
* `static/`: CSS and image assets.
